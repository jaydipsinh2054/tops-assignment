Question 3

Order Relationship

type Order {

  id: ID!
  products: [Product!]!
  total: Float!

}

type Query {

  getOrder(id: ID!): Order

}

Sample Query

query {

  getOrder(id: "1") {

    id
    total

    products {

      name
      price

    }

  }

}

Question 4

Introspection Discoveries

  • Query type is automatically available as the root query type.
  • Built-in scalar types such as String, Boolean, Float, and ID are included automatically.
  • GraphQL provides built-in introspection types like __Schema, __Type, and __Field, even though they were not explicitly defined.
