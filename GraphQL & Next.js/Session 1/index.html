Question 1

Fetch the name and image of any 3 Pokémon.

  pokemon_v2_pokemon(limit: 3) {
    name
    pokemon_v2_pokemonsprites {
      sprites
    }
  }
}

Question 5

GraphQL Mutation

mutation {
  addPlaylist(
    name: "Workout Mix"
    description: "Songs for the gym"
  ) {
    id
    name
    description
  }
}
Explanation
Part Description
mutation Starts a data modification operation.
addPlaylist Creates a new playlist.
name Name of the playlist.
description Description of the playlist.
id Returns the new playlist ID.
name Returns playlist name.
description Returns playlist description.
