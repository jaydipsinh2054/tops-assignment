Schema Structure (Introspection)

Query

├── playlist(id: ID!): Playlist
├── songs: [Song]

Playlist

├── id: ID!
├── name: String!
├── songs: [Song]

Song

├── id: ID!
├── title: String!
├── artist: String!
├── duration: Int!

You can explore the schema in Apollo Sandbox or GraphiQL using the Schema Explorer or by running the introspection query.
