Question 4

Role of ApolloProvider

  • ApolloProvider connects Apollo Client with the React application.
  • It uses React Context to share the Apollo Client instance with all child components.
  • Any component inside ApolloProvider can use hooks like useQuery and useMutation without creating a new client.
  • This avoids passing the client as props to every component.
