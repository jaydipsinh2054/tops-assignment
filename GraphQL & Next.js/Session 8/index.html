Question 5

SSR vs SSG Diagram

           SSG (getStaticProps)

 Build Time
      |
Generate HTML
      |
Store Static Page
      |
 User Requests Page
      |
Instant Response


--------------------------------------


      SSR (getServerSideProps)

User Requests Page
        |
Server Fetches Data
        |
Generate HTML
        |
Send Response

Explanation

Static Site Generation (SSG) creates pages during the build process, making them very fast and suitable for content that changes infrequently. Server-Side Rendering (SSR) generates pages for every request, ensuring users always receive the latest data. SSG is ideal for blogs or movie lists, while SSR is better for personalized pages like user orders.
